# CityExplorer

Swift learning app built using the SwiftUI framework
![](CityExplorer/Demo/CityExplorerDemoGIF.gif)

### Utilizes the following features:
- MapKit
- CoreLocation
- Yelp API
- UIKit Views
- NavigationView/NavigationLink
- ScrollView
- TabBar
- Lazy Stacks
- EnvironmentObject
- State
- TabView
- MVVM
- Remote JSON Data
- JSON Decoding
- Do/Catch
- Guard

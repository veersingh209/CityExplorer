# CityExplorer

Swift app to show highest rated destionions near users current location. Data gathered from yelp

![](CityExplorer/Demo/CityExplorerDemoGIF.gif)

### Utilizes the following features:
- MapKit
- CoreLocation
- Yelp API
- UIKit Views
- NavigationView/NavigationLink
- ScrollView
- TabBar
- Lazy Stacks
- EnvironmentObject
- State
- TabView
- MVVM
- Remote JSON Data
- JSON Decoding
- Do/Catch
- Guard
